About This Application

  • Machine learning-based system for accurate house rent prediction
  • Real-time prediction using trained regression models
  • Considers location, area, BHK, and furnishing features
  • Provides market insights using city & BHK-level median comparison
  • Classifies properties as Cheap, Moderate, or Expensive
  • Developed RESTful API for real-time ML-based prediction
  • Implements an end-to-end ML pipeline from data preprocessing to deployment
  • Integrates map-based location visualization for better spatial understanding
  • Provides data-driven insights for better rental decision-making
